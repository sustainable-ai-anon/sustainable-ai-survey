# Sustainable AI Survey – Analysis Code

This repository contains the R code used to generate the figures and analyses for the paper:

**"Sustainable AI research? Researchers’ perceptions of the environmental impact of AI and emissions-feedback tools – a mixed methods study"**

---

## Overview

This repository provides scripts to reproduce key visualizations from a survey on resarchers' attitudes toward the environmental impact of their work, sustainable AI, and emission-feedback tools as sustainable AI practice. The analyses focus on Likert-scale responses that assessed researchers' attitudes toward emission-feedback tools and their distribution across the full sample and demographic subgroups (other parts of the survey were analyzed qualitatively).

## How to Run the Code

1. Clone or download this repository
2. Place your dataset (if available) in:
   data/Analysis_for_R.csv
3. Run the scripts in the following order:

source("01_Figure3 - diverging_likert_plot.R
source("02_Appendix - Faceted_demographics_analysis.R")

## Outputs

The scripts generate the following figures in the /figures folder:

- Figure 3  
  Diverging stacked bar plot of Likert-scale responses, including a separate panel for "no opinion"

- Figure 4a  
  Faceted plot by gender

- Figure 4b  
  Faceted plot by age group

- Figure 4c  
  Faceted plot by highest education level

Figures are saved as high-resolution .png files (300 dpi), suitable for publication.

---

## Notes on Analysis

- Likert-scale responses are treated as ordered categorical variables
- "No opinion" responses:
  - Are visualized separately in Figure 3
  - Are excluded from subgroup analyses (Figure 4)
- Percentages are calculated within each survey item
